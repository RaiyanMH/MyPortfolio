Tech stack: Next.js (App Router) + Tailwind CSS + Framer Motion

Getting Started
---------------

1) Install dependencies

```bash
npm install
```

2) Run the dev server

```bash
npm run dev
```

3) Open `http://localhost:3000`

Content To Provide
------------------

- Projects: titles, descriptions, tech stacks, GitHub URLs, and images/screenshots
- CV: upload your PDF to `public/cv/Raiyan_Haque_CV.pdf`

Deployment
----------

1) Initialize git and push to GitHub

```bash
git init
git add .
git commit -m "feat: portfolio initial version"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```

2) Deploy to Vercel

- Import the GitHub repo on Vercel
- Use the default Next.js settings
- Your site will deploy to a URL like `https://raiyan.vercel.app` (configure custom domain as desired)
